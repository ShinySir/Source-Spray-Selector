Source Spray Selector

Source Spray Selector is a simple desktop application GUI written in Rust for previewing and selecting VTF spray files for Source Engine Games Like Team Fortress 2 and Garry's Mod. It works on Linux and Windows.

Features

  • Preview VTF spray images
  • Select VTF spray images in application to select them in-game
  • Favorite VTF spray images
  • Save and Load Game Profiles
